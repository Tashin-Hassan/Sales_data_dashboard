📊 Sales Analysis Dashboard (Power BI)
📌 Project Summary

Developed an interactive Sales Analysis Dashboard using Microsoft Power BI to analyze business performance across time, regions, customer segments, and product categories.

The solution transforms raw transactional data into actionable insights, enabling stakeholders to monitor KPIs, identify trends, and support data-driven decision-making.

🎯 Key Business Objectives
Track overall sales and profit performance
Identify high-performing regions and products
Analyze customer segments contribution
Monitor sales trends over time
Evaluate the impact of discounts on profitability
